## A list of open-source software tools for microscopy. 
I don't have experience with most of these packages; I just wanted to make a comprehensive list.

### Hardware control
- [micro-manager](https://github.com/micro-manager/micro-manager)
- [Storm control](https://github.com/ZhuangLab/storm-control)
- [ClearControl](https://github.com/ClearControl)
- [InControl](https://github.com/Huanglab-ucsf/InControl)
- [ACQ4](https://github.com/acq4/acq4)
- [York lab tools](https://github.com/AndrewGYork/tools)
- [microscope](https://github.com/MicronOxford/microscope)
- [OpenStage](https://github.com/raacampbell/openstage)
- [NicoLase](https://github.com/PRNicovich/NicoLase)

### Image display / processing
- [Imagej](https://github.com/imagej/imagej)
- [Icy](https://github.com/Icy-imaging)
- [flika](https://github.com/flika-org/flika)
- [pyqtgraph](https://github.com/pyqtgraph/pyqtgraph)
- [SIMA](https://github.com/losonczylab/sima)

### Modeling
- [MicroscPSF](https://github.com/MicroscPSF)
